Connection Guide
ANDROID HOST

Gunakan aplikasi PowerTunnel atau Virtual Host. Masukkan IP host ke dalam daftar aplikasi.

IOS HOST

Gunakan aplikasi Surge 5. Masukkan IP host melalui menu konfigurasi Surge untuk tunneling.

VIRTUAL HOST

Cara instan tanpa aplikasi tambahan. Langsung buka portal melalui browser untuk akses otomatis.

WINDOWS HOST

Edit file hosts di C:\Windows\System32\drivers\etc. Tambahkan IP dan nama server.
